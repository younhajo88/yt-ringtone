# 유튜브 벨소리 만들기

Windows PC에서 실행하고, PC 또는 휴대폰 브라우저로 접속해 유튜브 영상의 오디오를 29초 MP3 벨소리 파일로 만드는 개인용 웹 도구입니다.

## 주요 기능

- 유튜브 검색어로 영상 검색
- 유튜브 URL 직접 입력
- 검색 결과에 제목, 채널명, 영상 길이, 썸네일 표시
- 오디오 미리듣기
- 시작 시간 선택
- 29초 MP3 클립 생성
- 브라우저에서 완성된 MP3 다운로드

## 준비물

이 프로그램은 아래 도구가 Windows PC에 설치되어 있어야 합니다.

- Node.js 20 이상
- yt-dlp
- ffmpeg
- ffprobe

설치 여부 확인:

```powershell
node --version
yt-dlp --version
ffmpeg -version
ffprobe -version
```

## 설치

```powershell
npm install
```

## 실행

```powershell
npm start
```

또는 Windows에서 `start-ringtone.bat` 파일을 더블클릭하면 서버가 실행되고 브라우저가 자동으로 열립니다.

기본 주소:

```text
http://127.0.0.1:8001
```

같은 네트워크의 휴대폰에서 접속하려면 Windows PC의 IPv4 주소를 확인합니다.

```powershell
ipconfig
```

예를 들어 PC IP가 `192.168.0.10`이면 휴대폰 브라우저에서 아래 주소로 접속합니다.

```text
http://192.168.0.10:8001
```

## 사용 방법

1. 검색어 또는 유튜브 URL을 입력합니다.
2. 검색 결과에서 원하는 영상을 선택합니다.
3. 오디오가 준비되면 미리듣기로 원하는 지점을 찾습니다.
4. 시작 시간을 조정합니다.
5. `29초 벨소리 만들기`를 누릅니다.
6. 생성된 MP3 파일을 다운로드합니다.
7. 휴대폰에서 다운로드한 파일을 벨소리로 지정합니다.

## 제한사항

- 30분을 초과하는 영상은 준비하지 않습니다.
- 플레이리스트 전체 변환은 지원하지 않습니다.
- Android/iPhone 벨소리 자동 지정은 지원하지 않습니다.
- 로그인 기능은 없습니다. 외부 네트워크에 포트를 열면 접속 가능한 사람이 앱을 사용할 수 있습니다.
- 휴대폰에서 접속하려면 Windows 방화벽에서 TCP `8001` 포트를 허용해야 할 수 있습니다.
- 유튜브 다운로드 가능 여부는 `yt-dlp`, 유튜브 정책, 네트워크 상태에 영향을 받을 수 있습니다.

## 환경 변수

필요하면 실행 전에 환경 변수를 지정할 수 있습니다.

| 이름 | 기본값 | 설명 |
| --- | --- | --- |
| `HOST` | `0.0.0.0` | 서버가 바인딩할 호스트 |
| `PORT` | `8001` | 서버 포트 |
| `DATA_DIR` | `data` | 다운로드/생성 파일 저장 폴더 |
| `MAX_SOURCE_DURATION_SECONDS` | `1800` | 허용할 최대 원본 영상 길이 |
| `YTDLP_COMMAND` | `yt-dlp` | yt-dlp 실행 파일 이름 또는 경로 |
| `FFMPEG_COMMAND` | `ffmpeg` | ffmpeg 실행 파일 이름 또는 경로 |
| `FFPROBE_COMMAND` | `ffprobe` | ffprobe 실행 파일 이름 또는 경로 |

예시:

```powershell
$env:PORT = "4000"
npm start
```

## 파일 저장 위치

기본적으로 앱은 `data/` 폴더 아래에 작업 파일을 저장합니다.

- `data/source/`: 다운로드한 원본 오디오
- `data/preview/`: 미리듣기용 파일
- `data/clips/`: 생성된 29초 MP3 파일

`data/` 폴더는 Git에 포함하지 않습니다.

## 사용 범위

이 도구는 개인용으로 만든 로컬 웹앱입니다. 본인이 다운로드하고 변환할 권리가 있는 콘텐츠에 사용하는 것을 전제로 합니다.
